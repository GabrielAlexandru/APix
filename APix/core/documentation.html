Abstract

APix is a web application which offers its users te tools to draw together online. This document describes how APix works.

About

Users are able to:

  • - sign up an account
  • - log in
  • - draw using multiple drawing tools on a canvas
  • - share their canvas with other users connected to the app
  • - chat with other users
  • - draw on desktop using a remote(android phone)

Implementation

The backend is implemented using the python-based framework Django. Plugins included in the project: django-registration, channels(websockets).

Frontend is pure javascript, css3 and html5.

The sign up form is pretty simple, a new user needs a username, email and a password. The form will display help text for input and errors in case there are any (username already used by someone else for example).

The user will be redirected to this sign up form page on him first accesing the web-application. Route: /apix/register.

Canvas & Controls

On a successful sign-up or login, the user will be redirected to this page (otherwise the user will be redirected back to the login/signup page). To draw on the canvas, the user can just start click-moving the mouse on it.

Route: /apix/draw

The user can use the controls on the left side of the page to:

  • - logout
  • - connect to someone else's room and draw
  • - chat with people in the current room
  • - use the tools available to draw, download, or manipulate the canvas

Tools

Color Selection, the user can select one of seven preset colors or pick a unique one to draw with.

The selected color will appear on the size slider thumb.

The user can use the slider to change the pencil size.

Shape Drawing. The user can draw a circle or a rectangle on the canvas

Canvas Picker. With this tool the user can select a color from the canvas and set it as the drawing color.

Clear Canvas. With this tool the user can erase the entire canvas.

Capture. With this tool the user can capture the current state of the canvas and the image will be shown below the tools. This was intended to be used for GIF maker, but we can't make GIFs without libraries, and we want 10.

Download PNG. With this tool the user can capture the current state of the canvas and download it in PNG format.

Download SVG. With this tool the user can capture the current state of the canvas and download it in SVG format.

Upload Image. With this tool the user can upload any image from his computer and put it on the canvas.

Remote Control

A user can connect with his android phone and start drawing on the canvas that is visible on the desktop by moving the phone around

There is a special route for this functionality and the user will be redirected automatically if the user-agent is an android phone. Route: /apix/draw-mobile
